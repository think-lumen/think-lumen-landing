Under the hood

How the three MCPs connect

Each data source has a specific job. Claude orchestrates all three so you never have to switch context.

Morningstar pulls the fundamentals

Star ratings, fair value estimates, moat ratings, analyst forecasts, and attribution data — the analytical backbone of every decision.

Source: Morningstar MCP

Aiera monitors the calendar

Earnings call transcripts, event alerts, guidance changes, and overnight developments — so nothing slips through before the open.

Source: Aiera MCP

Claude synthesizes everything

Combines both data sources into plain-language analysis, prioritized alerts, and ranked trade recommendations tailored to your book.

Source: Claude AI

Egnyte stores the output

Reports, models, and notes are saved directly to your firm's file storage — version-controlled, organized, and ready to share with clients.

Source: Egnyte MCP
Getting started

Set up in under 10 minutes

Install the plugins, connect your data subscriptions, and you're live. No code, no infrastructure — just markdown files.

1 · Install the plugins

Add the marketplace and install the core plugin first, then the wealth management add-on.

claude plugin install financial-analysis
claude plugin install wealth-management

2 · Authenticate data sources

Each MCP requires a subscription. Authenticate once and Claude handles the rest.

/morningstar:authenticate
/aiera:authenticate
/egnyte:authenticate

3 · Run your first morning scan

Just tell Claude your holdings and ask what you need to know. No templates, no setup.

"What do I need to know about my book today? Holdings: MSFT, AAPL, JPM..."

4 · Customize for your firm

Drop your IPS targets, client names, and report templates into the skill files. Claude learns your process.

Edit skills/portfolio-rebalance/SKILL.md
Edit skills/client-report/SKILL.md
